myFlix Angular App

Angular app that represents the frontend of a restful API. Internet movie database that allows registered users to browse movies by Title, Genre, Director. Users can change their username, email address and birthday. While browsing the movies collection, users can add movies to their list of favorites. The interface views handle data through previously-defined REST API endpoints (see Movie API in the Project section).

App Features:

  • App displays a welcome view where users will be able to either log in or register an account.
  • Once authenticated, the user will be able to view all movies.
  • Movie cards have options to view the movie synopsis, details about the director, and a description of the movie’s genre.
  • Users can click the “heart” icon on the movie card, adding it to their list of favorites
  • Users can view their personal profile, where their list of favorite movies will be displayed
  • The profile page also allows users to update their information (username, birthday, email), or delete their account

Technologies Used:

  • Angular CLI
  • Typescript
  • Node.js
  • Angular Material
  • Compodoc
